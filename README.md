# RPAライセンス料 & 構築費用診断ツール (移行シミュレーター)

本ツールは、導入規模や移行対象ロボットのボリューム（難易度別の個数）に基づき、UiPathからPower Automate (PAD) へのライセンス移行による3年間の総削減コスト（TCO）を試算・比較するシミュレーションツールです。

## 主な機能
- **ライセンス費用比較**: 開発者ライセンス、有人ロボット（Attended）、無人ロボット（Unattended）、Orchestratorサーバー費用のシミュレーション。
- **削減効果の自動算出**: 3年間での合計削減金額を可視化。50万円以上の削減で金額を強調表示。
- **技術障壁の自動診断**: Orchestratorの利用有無や高難易度ロボット数に応じた「移行の障壁」を動的に警告。
- **フリーペーパー請求導線**: フォーム送信により「移行成功ガイドブック（フリーペーパー）」を自動生成・ダウンロード可能。

## 起動手順
```bash
streamlit run app.py
```
